## 🚀 My Featured Projects

### 🎬 Quick-Shake - 短视频聚合与推荐平台
A web-based short video platform providing core services like video uploading, viewing, personalized recommendations, multi-level comments, and real-time chat.

**Key Highlights:**
- ✨ **High-Performance Review Architecture**: Designed a layered review system with a custom thread pool, boosting processing capacity to **300 reviews/minute**.
- 🚀 **Optimized Recommendation Engine**: Implemented a user interest model with Redis ZSet and pipelines, increasing the recommendation API's QPS from **80 to 180**.
- ⚡ **Ultra-Fast Comment System**: Achieved a response time of just **45ms** for 10,000-level comment threads through optimized table design and hierarchical indexing.
- 🔥 **Efficient Hot-List Generation**: Reduced real-time computation by **95%** using SpringTask and TopK algorithms for scheduled updates of trending videos.

**Tech Stack:** `SpringBoot`, `MySQL`, `Redis`, `MyBatis`, `七牛云 (Qiniu Cloud)`

**[🔗 查看项目源码 (Gitee)](https://gitee.com/ZSQAQ666/quick-shake)**

### 🤖 AgentX Platform - 零学习成本的个性化AI Agent平台
A full-stack application dedicated to building personalized AI Agents with zero learning cost, driven by natural language.

**Key Highlights:**
- 🌐 **DDD-based Core Architecture**: Manages the entire Agent lifecycle (creation, versioning, deployment) and supports multi-agent collaboration via OpenAPI.
- 🧠 **Advanced RAG Framework**: Integrates vector and full-text search with a Reranker model, significantly enhancing search accuracy and user experience.
- 🛠️ **Deeply Modified LangChain4j**: Enhanced the framework to support preset parameters for tools, enabling more complex and predictable agent behaviors.
- 🛡️ **Self-Developed High-Availability Service**: Provides robust load balancing, circuit breaking, and rate limiting for integrated LLMs, ensuring service stability.

**Tech Stack:** `JDK17`, `SpringBoot`, `DDD`, `PostgreSQL`, `LangChain4j`
